This is a Galactic Intelligence Network Graph developed by Next.js and D3.js.

## Running Application

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
